üìã Entitas Eksternal

1. Super Admin

Deskripsi: Pengguna dengan akses penuh ke admin panel, termasuk manajemen user admin.

Aktivitas: Login, CRUD Dokter, CRUD Layanan, CRUD FAQ, CRUD Halaman Konten, Kelola User Admin, Pengaturan Akun
Alur Data:
Input: Kredensial login, data operasi CRUD (dokter, layanan, FAQ, konten, user admin)
Output: Dashboard, konfirmasi operasi, data yang dikelola, info user admin

2. Admin

Deskripsi: Pengguna dengan akses terbatas ke admin panel, tidak dapat mengelola user admin.

Aktivitas: Login, CRUD Dokter, CRUD Layanan, CRUD FAQ, CRUD Halaman Konten, Pengaturan Akun
Alur Data:
Input: Kredensial login, data operasi CRUD (dokter, layanan, FAQ, konten)
Output: Dashboard, konfirmasi operasi, data yang dikelola

3. Pengunjung Website (Public Visitor)

Deskripsi: Pengunjung yang mengakses website publik untuk melihat informasi klinik.

Aktivitas: Lihat website, ganti bahasa, isi kuesioner
Alur Data:
Input: Request halaman, pilihan bahasa, response kuesioner
Output: Konten website yang dilokalisasi sesuai bahasa

4. Google Translate API (Layanan Eksternal)

Deskripsi: Layanan eksternal untuk auto-translation dari bahasa Indonesia ke bahasa Inggris.

Alur Data:
Input: Teks bahasa Indonesia
Output: Teks bahasa Inggris

üíæ Penyimpanan Data (Internal)

Sistem menggunakan database untuk menyimpan:

  • admin_users - Data admin user
  • doctors - Data dokter
  • services - Data layanan medis
  • faqs - Data FAQ
  • content_pages - Konten halaman (Tentang Kami, Kontak, dll)
  • activity_logs - Log aktivitas admin
  • sessions - Data session

‚ö†Ô∏è Catatan Penting

  • Autentikasi: Admin harus login terlebih dahulu sebelum mengakses admin panel
  • Otorisasi: Super Admin memiliki akses penuh, Admin memiliki akses terbatas
  • Auto-Translation: Hanya admin yang input bahasa Indonesia, sistem otomatis translate ke bahasa Inggris
  • Lokalisasi: Website publik menampilkan konten sesuai bahasa yang dipilih pengunjung
  • Pencatatan Aktivitas: Semua aktivitas admin dicatat untuk audit trail
