# Panduan Membuka File Draw.io

File-file Draw.io (.drawio) yang telah dibuat dapat dibuka menggunakan:

## Cara Membuka File Draw.io

### 1. **Online (Recommended)**
- Kunjungi: https://app.diagrams.net/ atau https://www.draw.io/
- Klik **"Open Existing Diagram"**
- Pilih file `.drawio` dari folder `docs/`
- File akan terbuka dan siap diedit

### 2. **VS Code**
- Install extension **"Draw.io Integration"**
- Klik kanan file `.drawio`
- Pilih **"Open Preview"**

### 3. **Desktop App**
- Download Draw.io Desktop dari: https://github.com/jgraph/drawio-desktop/releases
- Install aplikasi
- Buka aplikasi dan pilih **"Open Existing Diagram"**
- Pilih file `.drawio`

### 4. **Google Drive / OneDrive**
- Upload file `.drawio` ke Google Drive atau OneDrive
- Klik kanan file dan pilih **"Open with" > "Draw.io"**

## File yang Tersedia

1. **CONTEXT_DIAGRAM_DRAWIO.drawio** - Diagram Konteks Sistem
2. **DFD_LEVEL0_DRAWIO.drawio** - DFD Level 0
3. **DFD_LEVEL1_DRAWIO.drawio** - DFD Level 1

## Catatan

- File Draw.io menggunakan format XML
- File dapat diedit dan disimpan secara langsung di Draw.io
- File dapat diekspor ke berbagai format: PNG, SVG, PDF, dll
- Struktur dasar sudah dibuat, Anda dapat menambahkan detail lebih lanjut sesuai kebutuhan

## Fitur Draw.io yang Berguna

1. **Shape Library**: Gunakan library yang tersedia untuk DFD
2. **Connectors**: Gunakan connector untuk alur data
3. **Labels**: Tambahkan label pada setiap alur data
4. **Styling**: Gunakan color palette untuk membedakan elemen
5. **Layers**: Organisir diagram menggunakan layers
6. **Export**: Ekspor diagram ke berbagai format
